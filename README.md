## 项目结构：
● NEPS端：公众监督员端。完成公众监督空气质量的全部业务流程。
● NEPG端：AQI检测网格员端。完成对公众监督员反馈的空气质量问题做AQI确认的全部业务流程。
● NEPM端：系统管理端。完成对公众监督员反馈的空气质量问题做指派处理，并对AQI检测网格员提交的AQI数据做统计处理，用于给决策者决策的依据。
● NEPV端：可视化大屏端。将系统管理端统计数据用可视化的方式展示给决策者。
